## Run locally

Open `index.html` in a browser, or use a local server:

```bash
npx serve .
```

## Deploy on Vercel

1. Push this repo to GitHub (see below).
2. In [Vercel](https://vercel.com), import the GitHub repo.
3. Leave build settings as default (no build command; output = root).
4. Deploy.

## Push to GitHub

If you haven't already:

1. Create a **new repository** on GitHub (e.g. `eliane-aunaturel` or `aibuddy-v1`). Do **not** add a README or .gitignore (you already have them).

2. In your project folder, add the remote and push:

```powershell
cd "c:\Users\Marsan\Desktop\Coding\aibuddy v1"
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git push -u origin main
```

Replace `YOUR_USERNAME` and `YOUR_REPO_NAME` with your GitHub username and repo name.
